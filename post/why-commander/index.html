Why commander?

Creating your own command line tool is amazing and can give your the opportunity of explore a new world out of the web but some times you will encounter some issues on creating this command line tools because the lack of reusability of that code that could be reused in each and every one of your tools and also the problem on sharing that code with others so that other people can have the opportunity of taking advantage of my work. Because of all that Commular was designed and developed to to implement a pluggable system of easy development without having to invent anything; it’s based on commander new increasing productivity and code reusability.
