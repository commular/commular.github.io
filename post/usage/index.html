Usage:

To use it in your command line you only need to:

  1. Create your project folder.
  2. Create the package.json file.
npm init
  1. Install Commular locally.
npm install commander --save
  1. Create a bin folder in the root of your project folder.
  2. Create a index.js file in the bin folder with the following code:
#!/usr/bin/env node
var commular = require('commular');
commular();
  1. Add the bin property to your package.json.
  "bin": "./bin/index.js"

To get more information I recommend to read the use cases in tests.
